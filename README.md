# 颜色格式转换工具 / Color Format Converter

单页面 React 应用，支持 HEX、RGB、HSL 三种颜色格式互转，实时预览，一键复制。

A single-page React app for HEX, RGB, HSL color format conversion with live preview and one-click copy.

**在线体验 / Demo**：<https://dsun6652-oss.github.io/easy-color-tool>

## 功能 / Features

- **格式互转**：HEX ↔ RGB ↔ HSL 任意格式输入，自动转换 / **Format conversion**: Input any format, auto-convert
- **拾色器**：点击色块打开原生拾色器手动选色 / **Color picker**: Click color block to pick
- **实时预览**：输入任意格式，立刻显示对应颜色块 / **Live preview**: Instant color display
- **一键复制**：每个格式结果单独复制按钮 / **One-click copy**: Copy each format separately
- **清空 / 重置**：清空所有输入，或重置为示例颜色 / **Clear / Reset**: Clear all or reset to example
- **深色模式**：默认深色主题 / **Dark mode**: Default dark theme
- **响应式布局**：适配移动端与桌面端 / **Responsive**: Mobile & desktop
- **中英文切换** / **Chinese & English**

## 支持的格式 / Supported Formats

- **HEX**：`#fff`、`#ffffff`
- **RGB**：`rgb(255, 255, 255)`
- **HSL**：`hsl(0, 0%, 100%)`
